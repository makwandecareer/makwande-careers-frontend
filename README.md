# Makwande Careers Frontend v3 Professional

Professional standalone Next.js frontend for the Makwande Careers API v6.

## Run

```cmd
cd /d E:\Makwande_Careers_Frontend
copy .env.example .env.local
npm install
npm run dev
```

Open http://localhost:3000

The original Makwande Careers logo is included in `public/makwande-careers-logo.jpeg`.
