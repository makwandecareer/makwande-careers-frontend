# Version 8 — 50 Formal Professional Templates

This release adds:

- 50 formal professional CV templates
- Headshot filters:
  - With photo
  - Without photo
- Column filters:
  - 1 column
  - 2 columns
- Template search
- Recommended template badges
- Client template selection
- Template selection saved in browser storage
- Seamless continuation to the CV Builder
- Responsive template marketplace

Install by replacing the current frontend folder, preserving `.env.local`, then run:

```cmd
cd /d E:\Makwande_Careers_Frontend
npm install
npm run dev
```
